6. 更新博客

每次写博客的时候，打开终端进入博客根目录，然后创建新的文章

文章写好后，一次执行以下命令，将改动同步到github上

  • hexo clean：清空public目录下的静态网页文件
  • hexo g：生成静态网页文件到public目录下
  • hexo d：将博客部署到github

7. 删除文章

1、删除文件夹source/_posts下目标文章markdown文件
2、删除.deploy_git文件夹
3、执行hexo clean后，再执行hexo g，hexo d 即可。
